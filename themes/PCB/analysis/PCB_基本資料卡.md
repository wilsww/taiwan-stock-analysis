| **股票代號** | 3037.TW（上市） |
| **成立時間** | 1990年 |
| **總部** | 桃園市龍潭區 |
| **員工數** | 約 20,000 人（含海外） |
| **產業定位** | 全球最大 ABF 載板廠；AI GPU 供應鏈最核心受益者 |

### 📅 關鍵催化劑

| 時間 | 事件 | 說明 |
|------|------|------|

### ⚠️ 關鍵風險追蹤

- **T-Glass 供應瓶頸（最核心）**：AGC/Corning 2026 無法及時擴產 → 欣興廣福廠先進 ABF 產能上限受限，即使需求旺盛也無法轉化為出貨成長
- **毛利率回升速度**：目前 15.8%，目標 20%+；ABF Film 成本轉嫁能力是關鍵
- **客戶集中度**：Intel/NVIDIA（間接），若 AI GPU 世代切換延後，訂單能見度下滑
- **Bear 情境壓力測試**：Hyperscaler CapEx YoY 降幅 >10% → AI GPU 採購縮減 → ABF 需求下修，超循環提前結束

- **Bear 情境壓力測試**：ABF ASP 漲幅低於預期 + 高層數認證延後 → 2026E EPS 下修至 NT$4–5 區間
