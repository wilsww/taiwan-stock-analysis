## 使用方式
在 Claude Code terminal 輸入：
```
請用以下框架分析 [公司名稱]（[代碼]）[季度] 財報：
```
然後附上以下模板內容。

## 標準財報分析框架

請依照以下 5 步驟，分析 {COMPANY}（{TICKER}）{QUARTER} 合併財報：

### 步驟 1：毛利率趨勢
- 本季合併毛利率：X%（QoQ ±Xpp，YoY ±Xpp）
- 若為華邦電（2344）：**必須拆分純記憶體段落**（排除新唐稀釋效果）
- 趨勢判斷：上行加速 / 持平 / 反轉向下
- 與超級循環歷史高點比較（南亞科：2021年~57%；華邦電：2021年~40%）

### 步驟 2：產能利用率與 ASP
- 產能利用率：X%（80%+ 為健康；<70% 為警戒）
- ASP 變化：QoQ ±X%，YoY ±X%
- 產品組合變化：DDR5/DDR4/Specialty 佔比移動
- 市場定價環境：合約價 vs 現貨價差

### 步驟 3：庫存水位（DSI）
- 公司端 DSI：X 天（健康：45-60 天）
- 客戶端庫存狀況：正常 / 偏高 / 去化中
- 與上季比較：QoQ ±X 天

### 步驟 4：資本支出
- 本季 CapEx：NT$X 億（QoQ ±X%）
- 全年指引：NT$X 億（是否有上修/下修）
- 性質判斷：擴產性 / 維護性 / 製程轉換
- 新廠/新節點時程確認

### 步驟 5：Guidance Tone
- 措辭評估：🟢 積極 / 🟡 中性 / 🔴 保守
- 管理層關鍵引述（中文原文）
- 下季展望與本模型預測差距
- 超預期/低於預期的項目清單
