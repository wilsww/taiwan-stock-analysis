## 2. 業務結構拆解

```
合併營收 100%
├─ Memory（CMS + Flash）~65–67%
│   ├─ CMS（Specialty DRAM，DDR4/LPDDR4）~30–32% — 景氣彈性大
│   └─ Flash（NOR Flash 全球第一、SLC NAND 第三）~35% — 穩定龍頭
└─ Logic IC（新唐 4919，為子公司）~33% — GM 35–42%，稀釋合併 GM ~9pp
```

**估值關鍵**：Memory 段才是循環 beta 來源；合併 EPS 被 Logic 稀釋，須分段推估。

## 3. 2026 營收推估

### Top-down

Q1'26 月營收 382.52 億（YoY +91.3%）。假設 Q2–Q4 QoQ +5–10%（DRAM 續漲、16nm/24nm 新製程 H2 放量）：

| 季度 | 營收（NT$ 億） |
|---|---|
| Q1'26 | 382.52 |
| Q2'26 | 402–420 |
| Q3'26 | 425–450 |
| Q4'26 | 450–480 |
| **全年** | **1,660–1,735** |

vs 2025 全年 894 億 → **YoY +86–94%**。

### 毛利率拆解

- Logic（新唐）GM ~38% × 營收佔比 33%
- Memory GM 2026 推升至 55–60%（DRAM 超級循環 + Flash 定價改善；落後南亞科 1–2 季轉換節奏）× 67%
- **合併 GM ≈ 0.67 × 57% + 0.33 × 38% = 50.7%**

### EPS 上限模型（樂觀）

```
營收 1,700 億 × GM 50% = 毛利 850 億
OpEx（~12.5% of rev）= 213 億
營業淨利 = 637 億（OpM 37.5%）
+ 業外 +10 億
– 稅（ETR 15%）= 97 億
– 少數股東權益（新唐非 100%，~11%）= ~55 億
= 歸母淨利 495 億
÷ 45 億股 = EPS ~NT$11.0
```

### 法人保守校準

考量以下衝擊，下修 GM 至 45%：

1. 16nm CMS 轉換期良率爬坡拖累 H1
2. 新唐工業 MCU 去化仍慢
3. 匯率 NT$ 升值侵蝕毛利 2–3pp
4. CapEx 跳升 → 2026–27 折舊 YoY +30% 侵蝕 GM ~3pp

```
毛利 765 億 – OpEx 213 億 = OpI 552 億
OpM 32.5% → 歸母淨利 ~420 億 → EPS ~NT$9.4
```

**2026E EPS 合理區間：NT$7.5–10.0**（法人共識 6.3–7.5 偏保守，反映新唐拖累與良率不確定性）

## 8. 關鍵風險

1. **16nm CMS 量產爬坡**：良率不如預期 → 毛利率無法跟上南亞科水準
2. **Flash 24nm 定價能力**：若未能高於舊製程 ASP → 折舊吃掉超循環紅利
3. **新唐 Logic IC 獲利穩定性**：工業 MCU 去化不利 → GM 跌破 35% 衝擊合併
4. **CapEx 421 億爆發**：2027 折舊壓力大增，若 2028 供給週期反轉 → 產能利用率壓力
5. **CUBE 專案商業化**：2027 才顯著貢獻，短期估值已部分透支此預期
6. **地緣風險**：中國 CXMT DDR4 衝擊華邦電 Specialty 市場

---

## 9. 結論

- **加權目標價 NT$109**（+24%）；風報比 1:4.5
- **建議**：**持有/加碼**。PBR 0.92x 位階低、DRAM 超循環紅利釋放剛開始、Flash 龍頭地位提供下方保護
- **建倉策略**：現價 87.8 在 Base 情境 -16.4% 以下，分批佈建合理；若回測 NT$80 附近（PBR ~0.85x）可加碼
- **主題定位**：與南亞科組成「DRAM 超級循環組合」—— 華邦防守 + 南亞進攻
