## 命名規範（建議）

| 類型 | 格式 | 範例 |
|------|------|------|
| 財報 PDF | `{ticker}_{company}_{period}_{type}.pdf` | `2408_南亞科_2026Q1_財報.pdf` |
| 法說簡報 | `{ticker}_{company}_{period}_法說_{lang}.pdf` | `8046_南電_2026Q4_法說_中文.pdf` |
| 美股 | `{ticker}_{form}_{period}.pdf` | `MU_10Q_2026Q2.pdf` |
| 分析檔 | `{theme}_{分析類型}.{ext}` | `DRAM_供應鏈全景圖.jsx` |
| 快照 | `{theme}_財務快照.md` | `DRAM_財務快照.md` |

**原則**：ticker 或 theme 開頭 → 排序時同主題相鄰 → 中英混用但順序固定。
